## Description

CLI typescript app
fetch & write data to .json file

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run dev -- -o filename -p // save all projects
or
$ npm run dev -- -o filename -p 1 2 3 <projects ids>...




# watch mode
$ npm run start:dev

# production mode
$ npm run build
$ npm run start:prod
```

### important

setup TOKEN value in .env
